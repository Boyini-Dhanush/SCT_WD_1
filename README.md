# 🎓 EduPath – Personalized Learning System

## SkillCraft Technology | Web Development Internship – Task 1

EduPath is a modern, responsive, and interactive landing page developed as part of the **SkillCraft Technology Web Development Internship**. The project is designed to provide a clean and engaging user interface for a personalized learning platform, demonstrating responsive web design principles and interactive front-end development using HTML, CSS, and JavaScript.



## 📖 Project Overview

EduPath aims to enhance the online learning experience by providing an intuitive and user-friendly interface. The landing page showcases the concept of a personalized learning platform with responsive layouts, interactive navigation, and modern UI elements suitable for desktop and mobile devices.



## ✨ Key Features

- 📱 Fully Responsive Design
- 🧭 Fixed Navigation Bar
- 🎨 Navigation Bar Color Change on Scroll
- 🌙 Dark Mode Toggle
- ☰ Responsive Hamburger Menu
- ✨ Smooth Scroll Animations
- 🔝 Back-to-Top Button
- 📩 Contact Form
- 🎯 Interactive Hover Effects
- 👣 Professional Footer


## 🛠️ Technologies Used

- **HTML5** – Page Structure
- **CSS3** – Styling & Responsive Design
- **JavaScript (ES6)** – Interactive Features
